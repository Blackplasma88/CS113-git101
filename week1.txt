ในภาษา c การเขียนโค้ดทุกครั้ง จะต้องการ include libraty เข้ามาก่อนเสมอ
ทำได้โดย #include ตามด้วยชื่อ library นั้นๆ

ในการรันแต่ละครั้ง ตัวคอมไพล์จะวิ่งไปหา main ฟังก์ชันเสมอ

library <stdio.h> มีไว้ในการรับ input output ทั่วๆไป
เช่น printf() ใช้แสดงข้อมูลออกทาง stdout
    scanf() ใช้รับข้อมูลทาง stdin
